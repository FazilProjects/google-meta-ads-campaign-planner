# Project Brief

## Project Name

Google & Meta Ads Campaign Planner Dashboard

## Owner

Fazil Waseem

## Purpose

Create a professional frontend dashboard that demonstrates practical campaign planning knowledge for Google Ads and Meta Ads. The project should be suitable for a portfolio and should help organize paid media thinking before campaign launch.

## Positioning

This is a portfolio project for Fazil Waseem, an aspiring Performance Marketer learning:

- Google Ads
- Meta Ads
- Lead generation
- Ecommerce marketing
- Service business marketing
- AI-assisted marketing systems

The dashboard should be honest about its scope. It is a planning interface, not a connected ad platform.

## Primary Users

- Portfolio reviewers
- Marketing mentors
- Potential clients or employers
- Fazil while practicing campaign planning workflows

## Core Workflow

1. Enter business brief and offer context.
2. Select campaign objective.
3. Review Google Ads planning direction.
4. Review Meta Ads planning direction.
5. Define target audience notes.
6. Review keyword ideas.
7. Draft ad copy.
8. Estimate budget allocation.
9. Check funnel readiness.
10. Check launch readiness.
11. Review KPI planning table.
12. Export the campaign plan.

## Campaign Concepts To Represent

- Business objective
- Audience
- Offer
- Budget
- Creative and ad copy
- Funnel stage
- Launch checklist
- KPIs and reporting

## Non-Goals

- No live Google Ads integration
- No live Meta Ads integration
- No account authentication
- No media buying automation
- No claim that generated plans guarantee performance
- No external frameworks or libraries

## Success Criteria

- The first screen feels like a real dashboard, not a marketing landing page.
- All requested sections exist.
- The UI is responsive on desktop and mobile.
- The copy stays honest and professional.
- The project can run from static files on GitHub Pages.
