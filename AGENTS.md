# AGENTS.md

## Project Context

This repository contains a static frontend portfolio project:

Google & Meta Ads Campaign Planner Dashboard

The project belongs to Fazil Waseem and is intended to demonstrate performance marketing planning knowledge through a professional dashboard UI.

## Scope Rules

- Keep the project static and GitHub Pages compatible.
- Use plain HTML, CSS, and JavaScript.
- Do not add external frameworks or libraries unless the owner explicitly asks.
- Do not present the app as a real Google Ads or Meta Ads platform.
- Do not imply official affiliation with Google, Meta, or any ads platform.
- Keep marketing claims honest and educational.

## Product Direction

The dashboard should help organize campaign planning around:

- Business objective
- Audience
- Offer
- Budget
- Creative and ad copy
- Funnel stage
- Launch checklist
- KPIs and reporting

## Design Direction

- Clean, premium dashboard UI.
- Light background with dark text.
- Use Fazil's accent colors carefully: blue, red, yellow, and green.
- Prefer cards, grids, badges, KPI blocks, tabs, and workflow sections.
- Keep the first screen focused on the usable dashboard.
- Maintain responsive layouts for desktop and mobile.

## File Guide

- `index.html` contains semantic dashboard structure.
- `style.css` contains the visual system and responsive behavior.
- `script.js` contains objective switching, budget updates, reset, and export logic.
- `docs/PROJECT_BRIEF.md` documents project strategy and scope.
- `docs/DESIGN_SYSTEM.md` documents visual standards.
- `docs/CONTENT.md` documents wording and claim boundaries.
